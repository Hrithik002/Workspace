# Workspace
SignUpPage
A professional signup and login page with:
Java Spring Boot backend
Vanilla JavaScript frontend
SQLite database for local credential storage
Client-side and server-side validation
Project structure
frontend/ - HTML, CSS, and JavaScript UI
backend/ - Spring Boot API and SQLite persistence
Requirements
Java 17+
Maven 3.9+
Run the app
Open a terminal in the backend folder.
Start the server with Maven.
Open the app in your browser at http://localhost:8080
The database file is created automatically in the backend folder as signup.db.
